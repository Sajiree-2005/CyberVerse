# CyberVerse - Inside a Computer

An interactive cinematic exploration of how computers think, remember, communicate, and defend themselves.

## Features

- **Interactive CPU Pipeline**: Watch how instructions flow through the processor
- **Memory Hierarchy Visualization**: Explore cache levels and RAM
- **Network Communication**: See how data travels across systems
- **Security Concepts**: Learn about vulnerabilities and defenses
- **Engaging Storytelling**: Beautiful animations and transitions
- **Knowledge Checks**: Test your understanding with interactive quizzes
